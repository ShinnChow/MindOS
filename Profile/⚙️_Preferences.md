# ⚙️ Preferences & Workflow

## 🤖 AI 交互偏好 (AI Interaction)
- **风格**: 简洁直接 (Concise & Direct)，避免废话
- **代码**: 优先给出完整代码或关键片段，注重最佳实践
- **语言**: 默认使用中文 (Chinese)，技术术语可保留英文
- **回答逻辑**: 先结论后解释 (Conclusion first)

## 🖥️ 开发环境 (Dev Environment)
- **OS**: macOS (主要开发), Linux (服务器部署)
- **Editors**:
  - VS Code (主力)
  - Cursor (AI 辅助)
  - Antigravity (Google Internal)
- **Terminal**: Warp

## 📦 工具链 (Toolchain)
- **笔记/知识库**: Notion, Obsidian
- **任务管理**: 滴答清单 (TickTick)
- **同步**: 坚果云 (Nutstore)
- **浏览器**: Chrome
