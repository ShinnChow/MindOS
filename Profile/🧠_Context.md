# 🧠 Global Context

## 📖 常用术语 (Glossary)
- **SOP**: Standard Operating Procedure (标准作业程序)
- **MCP**: Model Context Protocol (模型上下文协议)
- **Agent**: 自主智能体
- **IP**: Intellectual Property (此处指个人品牌/影响力)

## 🔗 关键关联 (Key Relations)
- **SOP Note**: 本项目，核心知识库
- **Configurations**: 系统与工具配置文件
- **Workflows**: 自动化工作流脚本与文档
