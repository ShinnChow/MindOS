# 关系

用于维护可复用的人际关系信息，帮助 Agent 在沟通与协作中更贴合你的真实语境。

## 目录结构

- `📇 关系总览.csv`（正式总览索引）
- `🧪 关系示例索引.csv`（示例索引）
- `家人/`
- `朋友/`
- `同学/`
- `同事/`
- `导师/`

示例文件放在各分类目录下，命名为 `🧪_example_xxx.md`。

## 使用说明

- 正式数据：先看 `📇 关系总览.csv`，再根据 `MdPath` 读取对应人物 `*.md`。
- 示例数据：参考 `🧪 关系示例索引.csv` 与各分类目录下 `🧪_example_xxx.md`，不要当作用户真实信息。
- 新增人物时，同时维护：`📇 关系总览.csv` + 人物 `*.md`。
- 字段规范与执行细则以 `INSTRUCTION.md` 为准。
