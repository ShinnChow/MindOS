# 🖥️ Server 常用软件

| 名称 | 类别 | 类型 | 安装方式 | 备注 |
|------|------|------|----------|------|
| Docker | 开发 | 容器 | `apt install docker.io` | |
| Docker Compose | 开发 | 容器 | `apt install docker-compose-plugin` | |
| Node.js | 开发 | 运行时 | `apt install nodejs` | 建议安装 LTS |
| Git | 开发 | 版本控制 | `apt install git` | |
| nvtop | 监控 | GPU 监控 | `apt install nvtop` | 需要 GPU |
| htop | 监控 | 进程监控 | `apt install htop` | |
| tmux | 系统工具 | 终端复用 | `apt install tmux` | |
| vim | 系统工具 | 编辑器 | `apt install vim` | |
| tree | 系统工具 | 文件查看 | `apt install tree` | |
| unzip | 系统工具 | 解压 | `apt install unzip` | |
| curl | 网络 | 请求工具 | `apt install curl` | |
| wget | 网络 | 下载工具 | `apt install wget` | |
